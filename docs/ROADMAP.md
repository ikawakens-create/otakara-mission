## ステップ2：ごほうび核（コンプリート演出＋ガチャ）

**目的**：毎日続けたくなる「ガチャの楽しみ」を載せる。

- [ ] 1日コンプリート判定 → ボーナス+1 ＆ スペシャル花丸演出（紙吹雪・音）

- [ ] 景品付与（ポイント/スタンプ/アイテム）と結果表示
- [ ] 同日二重付与の防止（gachaPulled）
