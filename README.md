# おたからミッション（アプリ版）

子ども（小学生・園児）向けの「おてつだい・生活習慣ミッション表」アプリ。
紙版で好評だったものをデジタル化し、**スタンプ・ガチャ・着せ替えコレクション**で
子どもが自分から続けたくなる仕組みにする。

- 対象ユーザー：娘2人（7歳/小2・6歳/年長）
- 使用端末：子どものAndroidスマホ・タブレット
- 配布形態：**PWA**（ブラウザで開いて「ホーム画面に追加」でアプリのように使える）
- バックエンド：**なし**（データは端末内に保存）

## いちばん大事な設計思想

**「コンテンツ（スタンプ・アイテム・ガチャの中身）はコードと分離する」**。
新しいスタンプやアイテムを足すときは、データ定義ファイルに1件追加するだけで済む構造にする。
これにより、機能を触らずに中身を無限に増やせる（＝アップデートが楽）。
